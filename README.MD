## 📋 What Is This?

This project contains a list of software and tools that help build digital public infrastructure. It focuses on public goods that anyone can use. This includes software for governance, civic tech, interoperability, and digital transformation.

The tools and resources here are open-source. That means anyone can view, use, and contribute to them. This makes it easier to build technology that benefits everyone, especially those in public sectors and development organizations.
